🛠️ Enterprise Renewal Copilot — starter

The roadmap says: this weekend, create the repo, a venv, a 50-record synthetic JSON, and a Python script that scores renewal risk. I scaffolded exactly that for you in this folder. Below is what's there and how to run it.

What I created next to this app

enterprise-renewal-copilot/
  README.md
  requirements.txt
  data/
    customers.json          # 50 synthetic records (no real data)
  backend/
    risk_report.py          # Week 1: reads JSON, scores renewal risk, writes report
    main.py                 # Week 2: FastAPI service (run: uvicorn backend.main:app --reload)
  tests/
    test_risk.py            # pytest starter

Run the Week 1 script

cd enterprise-renewal-copilot
python -m venv .venv
.venv\Scripts\activate        # Windows
pip install -r requirements.txt
python backend\risk_report.py     # -> data/renewal_report.json

Run the Week 2 FastAPI service

uvicorn backend.main:app --reload
# open http://127.0.0.1:8000/docs  (auto Swagger/OpenAPI)

Remember the roadmap rule: a completed ugly first version beats a magnificent diagram with no code. This starter is intentionally small so you can finish and extend it.
